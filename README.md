# cron
### A Speedy and Lightweight Cron Module

## Install
```
go get github.com/frisbm/cron
```

## Usage
```go
package main

import (
	"fmt"
	"github.com/frisbm/cron"
	"log"
)

func main() {
	cron, err := cron.Parse("* * * * *")
	if err != nil {
		log.Fatal(err)
	}

	now := cron.Now()
	fmt.Println(now) // true
}
```

See package documentation [here](https://pkg.go.dev/github.com/frisbm/cron)
